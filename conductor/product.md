# Initial Concept
Vexel is a high-performance LLM inference engine optimized for Apple Silicon (Metal) with a primary focus on providing developers with easy-to-integrate Go bindings for local LLM capabilities in their applications and services.

# Target Users
- **Application Developers:** Developers looking to integrate local LLM features into their Go-based apps and services with minimal friction.
- **LLM Kernel Developers:** Engineers optimizing Metal kernels who need a reliable harness for performance and correctness testing.
- **AI Infrastructure Engineers:** Engineers benchmarking different inference backends to ensure optimal resource utilization and performance.

# Primary Goals
- **High-Performance Metal Inference:** Deliver best-in-class performance on Apple Silicon by leveraging Metal-optimized kernels.
- **Seamless Go Integration:** Provide a native Go interface that makes it easy for developers to add LLM capabilities to their existing Go projects without complex CGO or external dependency management.
- **Correctness and Reliability:** Maintain a robust harness to ensure inference results match established industry standards like llama.cpp.
