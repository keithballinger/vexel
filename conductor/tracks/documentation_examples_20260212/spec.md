# Track Spec: Documentation & Examples

## Overview
Create comprehensive documentation and examples for Vexel, covering setup, usage (CLI and API), and integration into Go applications. This will improve adoption and ease of use.

## Goals
1.  **Usage Guide:** Clear instructions on how to run and configure Vexel.
2.  **API Reference:** Documentation for the `vexel/client` and gRPC/HTTP endpoints.
3.  **Examples:** `examples/` directory with code samples for common use cases.

## Technical Details
-   **README.md:** Update with detailed "Getting Started".
-   **examples/**: `simple-chat`, `batch-inference`, `streaming-server`, etc.
-   **Godoc:** Ensure public API has clear comments for `godoc`.
